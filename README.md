# Weather Forcast Site

A fully responsive, animated weather web application that delivers live weather forecasts for any location.

Here’s what it can do:

🔹 🌐 Location-Based Forecasts – Get weather by geolocation or by typing any place name (college, village, or city).

🔹 📡 Open-Meteo API + Nominatim Geocoding – No API key required!

🔹 📅 Current, Hourly, and Daily Weather – All neatly organized and displayed.

🔹 🌈 Weather Icons – Visual weather codes with emoji-based icons.

🔹 📥 Downloadable JSON – Export weather data in one click.

🔹 🎨 Smooth UI & Animations – GSAP-powered transitions for a sleek experience.

🔹 ⏰ Shows Local Time – Displays time based on searched location.

🔹 📱 Fully Responsive – Works great on mobile and desktop.
